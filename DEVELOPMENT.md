3. Create this file - DEVELOPMENT.md 
    ~~~~

    ~~~~
4. Create a virtual environment
    ~~~~
    virtualenv venv -p python3
    ~~~~
5. Create gitignore

    ~~~~
    touch .gitignore
    ~~~~
